## Problem Description

Write a Java program to print a character pattern like:
A
A B
A B C
A B C D
A B C D E

## Code

```java title="CharacterPattern.java"
public class CharacterPattern {
    public static void main(String[] args) {
        int rows = 5;

        for (int i = 1; i <= rows; ++i) {
            for (int j = 1; j <= i; ++j) {
                System.out.print((char)(j + 64) + " ");
            }
            System.out.println();
        }
    }
}
```

## Output

```bash
A 
A B 
A B C 
A B C D 
A B C D E 
```

## Explanation

1.  **ASCII**: `65` is 'A'. So `j + 64` gives 65 ('A') when `j` is 1.
2.  **Casting**: Cast the integer result to `char`.
